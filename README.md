# Arduino LED toggle switch on TTP223 touch sensor

## Bill of material
- Board: Arduino MEGA 2560  
- Resistor: 220Ω ±1%
```
| Module | Mode   | Pin |
|--------|--------|-----|
| TTP223 | INPUT  | 28  |
| LED    | OUTPUT | 24  |
```
